# Project Roadmap: StudySmart

This roadmap outlines the development phases for the capstone project over a 12-week duration.

### 🏗️ Phase 1: Foundation & Design (Weeks 1-2)
* [ ] Define JSON data structures for `users`, `notes`, and `groups`.
* [ ] Create high-fidelity wireframes in Figma.
* [ ] Initialize React project and folder architecture.
* [ ] Set up Tailwind CSS configuration.

### ⚙️ Phase 2: Core Functionality & Mock API (Weeks 3-5)
* [ ] Deploy **JSON Server** to handle CRUD operations for notes.
* [ ] Build the **Note-Exchange Hub** (Upload, List View, and Search).
* [ ] Implement the Upvote/Downvote logic using Fetch/Axios.
* [ ] Create User Authentication flow (Mock Login/Signup).

### 🤝 Phase 3: Collaborative Features (Weeks 6-8)
* [ ] Build the **Study Group Discovery** dashboard.
* [ ] Develop the **AI Study Matcher** logic (Frontend-based filtering).
* [ ] Create the **Live Study Room** UI (Chat interface and Whiteboard component).
* [ ] Integrate the **Pomodoro Timer** utility.

### 🧠 Phase 4: Enhancements & Optimization (Weeks 9-10)
* [ ] Implement **Focus Mode** (UI/UX state management).
* [ ] Build the **Low-Bandwidth** toggle (Conditional rendering for high-res assets).
* [ ] Add "Scholar Credit" badge logic for top contributors.
* [ ] Optimize mobile responsiveness across all views.

### 🧪 Phase 5: Testing & Launch (Weeks 11-12)
* [ ] Conduct User Acceptance Testing (UAT) with peer group.
* [ ] Fix UI/UX bugs and refine state transitions.
* [ ] Prepare final documentation and presentation slides.
* [ ] Deploy the frontend to Vercel/Netlify.
